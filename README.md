# 🐻 熊大快跑 ImGui 菜单源码

这是一个基于 ImGui 的安卓 JNI 原生菜单项目源码。仅供学习和交流 C++ 与逆向技术使用。

## 🛠️ 项目环境
* **语言**：C/C++
* **架构**：Android NDK (JNI)
* **界面库**：ImGui

## 🚀 编译说明
1. 本项目为纯 `jni` 源码结构。
2. 推荐使用 NDK 编译（如 `ndk-build`），或在安卓端使用 Termux / AIDE Pro 等环境进行编译。
3. 编译生成的 `.sh` 文件可直接注入或集成至熊大快跑应用中。
